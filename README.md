# Employee Tracker

Dự án **Employee Tracker** là một ứng dụng phần mềm giúp quản lý và theo dõi thông tin nhân viên một cách hiệu quả.

## 🚀 Tính năng chính
* Xem danh sách tất cả nhân viên.
* Thêm mới, cập nhật và xóa thông tin nhân viên.
* Quản lý thông tin phòng ban và vai trò của từng cá nhân.

## 🛠️ Cài đặt và Chạy dự án

**1. Clone dự án về máy**
git clone [https://github.com/hohquangsang/employ_tracker.git](https://github.com/username-cua-ban/employ_tracker.git)
cd employ_tracker

**2.Thiết lặp môi trường ảo**
python -m venv venv
# Trên Windows:
venv\Scripts\activate
# Trên macOS/Linux:
source venv/bin/activate

**3. Cài đặt thư viện**
pip install -r requirements.txt

**4. Chạy dự án**
python app.py
